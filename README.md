# CPP-Complex-Program

Building and Running:

To run the program above, navigate to the directory where the program file is located using your system's command line. Once in the program directory, follow the instructions below for your platform.
Windows – MinGW-w64

Compile the program:

		g++ bank.cpp -o bank

Run the program:

		bank

Linux – GNU C++ Compiler (g++)

Compile the program:

		g++ bank.cpp -o bank;

Run the program:

		./bank

macOS – Clang (Xcode Command Line Tools)

Compile the program using C++17 or above:

		clang++ -std=c++17 bank.cpp -o bank

Run the program:

		./bank
